1. **driver 主路径优先**：拦截器抛错被 InterceptorRunner 隔离，单个故障不影响 SSE 循环 / 兄弟拦截器 / 终态语义
2. **业务无关**：拦截器层 grep gate 落地（[test_interceptor_purity.py](../../tests/unit/test_interceptor_purity.py)），禁止 `vibe-trading|signal_engine|ma250|strategy|skill` 出现

4. **错误预算**：单拦截器累计错误 ≥10 静默 disable，写一次 logger.error

| 文件 | 角色 |

### 3.2 行为

| 维度 | 落地 |
|---|---|
| 订阅事件 | `initial_*_message` / `assistant_delta` / `tool_call_*` / `decision_received` |
| Assistant 增量 | 连续多片段 coalesce 成单条 message，遇 tool / decision 自动 flush |
| Slug 决议 | callback (sync/async) → 正则校验 → 命中则用 / 否则 fallback `untitled-<task_id[:6]>`；source 入 metadata |
| 敏感信息 | OpenAI / Stripe / Slack / AWS / GitHub key + 18 位身份证 + Bearer header 脱敏；幂等 |
| 截断 | 单条 content > 32 KiB 截断；总条数 > 2000 走 head + sentinel + tail |
| 落盘路径 | `<artifacts_dir>/<task_id>/conversations/<ISO8601>-<slug>.jsonl` |
| Artifact metadata | `subtype` / `conversations_path` / `slug` / `slug_source` / `message_count` / `truncated_count` / `terminal_status` |
